Git & GitHub

Git est un système de contrôle de version distribué qui permet de suivre les modifications apportées aux fichiers source pendant le développement logiciel. Il permet aux développeurs de collaborer efficacement, de restaurer des versions précédentes de leur code et de gérer les modifications de manière organisée.

GitHub est une plateforme de développement logiciel basée sur Git. Elle facilite la collaboration entre les développeurs en fournissant des fonctionnalités telles que le suivi des problèmes, la gestion des tâches, l'hébergement de projets, et bien plus encore. GitHub permet également de partager et de contribuer à des projets open source.
